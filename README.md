# Private VPN Node Toolkit

个人 VPN/代理节点搭建与管理工具集，基于 Claude Code Skills。

## 包含内容

- **vpn-node-setup** — 一键搭建指南，涵盖 Shadowsocks / Xray VLESS+Reality / Hysteria2 / VMess+WS+TLS+Nginx 四种方案
- **vpn-node-manage** — 日常管理：增删节点、查看日志、故障排查、安全加固、自动维护

## 适用场景

| 场景 | 服务器位置 | 用途 |
|------|-----------|------|
| 翻墙节点 | 境外 VPS (RackNerd/Bandwagon等) | 国内访问境外网站 |
| 回国节点 | 大陆 VPS (阿里云/腾讯云国内版) | 海外访问大陆限定内容 |

## 使用方法

将本项目导入 Claude Code 后，通过自然语言操作：

```
"帮我搭建一个 xray 节点"
"增加一个备用节点"
"节点连不上了帮我排查"
"设置自动维护"
"搭个回国节点"
```
